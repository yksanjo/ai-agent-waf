# Security Dashboard - Attack Overview

## Layout
- **Header**: "AI Agent WAF" branding, real-time indicator
- **Stats Bar**: Total attacks blocked, false positive rate, fraud prevented
- **Main Chart**: Attack volume timeline (last 7 days)
- **Attack Types**: Pie chart showing attack distribution
- **Recent Blocks**: Table of last 10 blocked attacks

## Key Elements
- Large counter: "2,847 attacks blocked today"
- Green/Red indicators for blocked/allowed
- Real-time attack stream (auto-updating)
- Geographic attack map (optional)
